House Rules

• Bullying

bullying of any kind will not be tolerated, this includes: ethnic discrimination (racism), derogatory comments towards a members religious beliefs, location (country/area of residence), sexuality or gender(sexism), derisive comments towards a members lack of knowledge or ability is also considered bullying. It is strongly encouraged that you report any member you suspect is bullying to admin, the offender will be swiftly removed from the group with a permanent ban.

• Advertisements

advertisements for your apps, YouTube channels, websites and blogs are only permitted on Sundays (GMT), if you attempt to post ads on any other day then we will consider this as spam. Repeated offenses will result in a ban. Any ad posted that's off-topic will be considered spam regardless of the day.

• Code Snippets

if you require help with source code then you are more than welcome to post code snippets so other member can help you. If your code snippet exceeds 10 lines then you must use sites like pastebin, hastebin, github gist etc and post the link. If you need help on how to do this please ask. We also advise not to post screenshots/photos of your code.

• Links In Comments

you may post a website, YouTube or blog link in a comment only if it's in direct response to the original post and serves to help them regarding their particular query/question, if the link does not satisfy this criteria then it will be deemed as spam, and will result in a ban.

• English Language Only

being an international group, we ask that you only post and comment in English. Any attempts to post in non-English will not be approved by admin. Repeatedly posting/commenting in non-English will result in a ban.

Happy coding :)
